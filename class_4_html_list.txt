HTML Lists:
	HTML lists allow web developers to group a set of related items in lists.

there are 2 type of list present in html

	1.unordered list
	2.ordered list

unordered list: 
	using <ul> tag then uder the <ul> we make a list using <li> tag

	ex:-
	<ul>
		<li>Subham</li>
		<li>Satyajeet</li>
		<li>Vikash</li>
	</ul>


ordered list:-
	using <ol> tag then uder the <ul> we make a list using <li> tag

	ex:-
	<ol>
		<li>Subham</li>
		<li>Satyajeet</li>
		<li>Vikash</li>
	</ol>


 Description Lists:-

 	1.HTML also supports description lists.

 	2.A description list is a list of terms, with a description of each term.

	3.The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term:


ex:-

<dl>

  <dt>Python</dt>
  	<dd>- It's a scrpting laguage</dd>
  	
  <dt>Java</dt>
 	 <dd>- It's object oriented laguage</dd>

</dl>


HTML List Tags
Tag	Description
<ul>	Defines an unordered list
<ol>	Defines an ordered list
<li>	Defines a list item
<dl>	Defines a description list
<dt>	Defines a term in a description list
<dd>	Describes the term in a description list
